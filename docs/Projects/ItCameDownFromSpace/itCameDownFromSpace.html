It Came Down From Space

Description

It Came Down From Space was the second game I made, together with 5 other students.

After making Invasion, we immediately got another project put in front of us to go make another game. This time, it needed to be inspired by an already existing game. By modifying that game by removing one critical feature and then we'd have to figure out how to still make the game great without that feature.
We chose Getting over it with Bennett Foddy and modified it by removing the feature of needing to go up, this game was our solution to not having that feature.
This game was made in C#, using the Monogame engine.
The game even got streamed on twitch, here is a video of a streamer playing it live!
