Dario's Carnage

Description

Dario's Carnage is the third game I made for a school project, the first in Unity.

Dario's Carnage is a chaotic first person shooter, which falls under the category of "boomer shooter". It is a result of my first experience with the Unity engine, together with 5 other students.
We needed to make multiple games for 1 week game jams and pick the best of these games to develop further. This is the game we picked, coming from the game jam theme "start with nothing".
