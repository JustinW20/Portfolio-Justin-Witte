Line to rectangle collision

As I have learn a lot after making this project, there is not a lot of code I'm proud of anymore. The only thing I'm proud of is the line-to-rectangle collision I made for a bossfight.
To make this, I first made line-to-line collision. As a rectangle consists of four lines, I could basicaly use line-to-line collision four times to get line-to-rectangle collision.
This looks as follows: 

                    
                            boolean lineLine(float x1, float y1, float x2, float y2, float x3, float y3, float x4, float y4) {

                                // calculate the direction of the lines
                                float uA = ((x4-x3)*(y1-y3) - (y4-y3)*(x1-x3)) / ((y4-y3)*(x2-x1) - (x4-x3)*(y2-y1));
                                float uB = ((x2-x1)*(y1-y3) - (y2-y1)*(x1-x3)) / ((y4-y3)*(x2-x1) - (x4-x3)*(y2-y1));
                            
                                // if uA and uB are between 0-1, lines are colliding
                                if (uA >= 0 && uA <= 1 && uB >= 0 && uB <= 1) {
                                  return true;
                                }
                                return false;
                              }

What this calculates is whether one of the two outer points of line A is between the two points of line B (uA). If the point is inbetween the two points of line B, the value of uA is between 0 and 1. It then does the same thing the other way around (uB). If both values are in the range between 0 and 1, the lines are colliding. Now I can use this for every side of the rectangle to make line-to-rectangle collision. 
                    
                        boolean lineRect(float x3, float y3, float x4, float y4, float xR, float yR, float xSize, float ySize) {

                            //checking if the line hits any of the players sides
                            boolean left = lineLine(x3, y3, x4, y4, xR, yR, xR, yR + ySize);
                            boolean right = lineLine(x3, y3, x4, y4, xR + xSize, yR, xR + xSize, yR + ySize);
                            boolean top = lineLine(x3, y3, x4, y4, xR, yR, xR + xSize, yR);
                            boolean bottom = lineLine(x3, y3, x4, y4, xR, yR + ySize, xR + xSize, yR + ySize);
                        
                            if (left || right || top || bottom) {
                              return true;
                            }
                            return false;
                        }
