Invasion

Description

Invasion is the very first game I made, together with three other beginning game development students.

At the very first day of the HBO-ICT Game Development study, I immediately got the great new that the first thing what we would be doing is make an actual game. Together with three other students I got thrown into the deep to make a game, which needed to be suitable for on-board entertainment system of an airplane, using the remote as a controller. Due to this we were limited in the controls we could use, comparable to an old snes controller.
This game was made using Java, in the Processing environment.
Together, we set out to find a unique mechanic we could fit into our game, to really make it stand out from the other games made during that project. We came up with the idea that the player would be unable to jump and it would need the knockback effects of the weapons it can use to progress through the levels.
As we were not very experienced and made mistakes causing loss of quality, we are planning to remake the game in the Unity engine!
